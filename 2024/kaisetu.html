第 1 問

座標空間内の点 $A(0,-1,1)$ をとる。$xy$ 平面上の点 P が次の条件 (i), (ii), (iii) をすべて満たすとする。

(i) P は原点 O と異なる。

(ii) $\angle AOP \ge \dfrac{2}{3}\pi$

(iii) $\angle OAP \le \dfrac{\pi}{6}$

P がとりうる範囲を $xy$ 平面上に図示せよ。

方針の立て方

① 状況理解で分かること

  • P $= (x,y,0)$ は $xy$ 平面上の点。$A = (0,-1,1)$ は $xy$ 平面の外。$O = (0,0,0)$。
  • 2つの角度条件を不等式に変換すれば、P の範囲が定まる。

② 用いた定石

「なす角の扱い」→ 内積と $\cos$ で条件を式にする（定石集FGver「§なす角の扱い」より）

解法

【条件 (ii) の式化】

$\angle AOP \ge \frac{2}{3}\pi$ を $\cos$ で書くと、$\cos(\angle AOP) \le \cos\frac{2}{3}\pi = -\frac{1}{2}$。

$$\cos(\angle AOP) = \frac{\vec{OA}\cdot\vec{OP}}{|\vec{OA}||\vec{OP}|} = \frac{(0)(x)+(-1)(y)+(1)(0)}{\sqrt{2}\cdot\sqrt{x^2+y^2}} = \frac{-y}{\sqrt{2}\sqrt{x^2+y^2}}$$

$\le -\frac{1}{2}$ より $\frac{y}{\sqrt{2}\sqrt{x^2+y^2}} \ge \frac{1}{2}$。両辺正なので2乗：

$$\frac{y^2}{2(x^2+y^2)} \ge \frac{1}{4} \quad \Longrightarrow \quad 2y^2 \ge x^2+y^2 \quad \Longrightarrow \quad y^2 \ge x^2$$

$y > 0$（途中で確認済み）と合わせて：$y \ge |x|$（$y > 0$ かつ $|y| \ge |x|$）。

【条件 (iii) の式化】

$\angle OAP \le \frac{\pi}{6}$ を $\cos$ で書くと、$\cos(\angle OAP) \ge \cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}$。

$$\vec{AO} = (0,1,-1), \quad \vec{AP} = (x,\, y+1,\, -1)$$ $$\cos(\angle OAP) = \frac{\vec{AO}\cdot\vec{AP}}{|\vec{AO}||\vec{AP}|} = \frac{(y+1)+1}{\sqrt{2}\sqrt{x^2+(y+1)^2+1}} = \frac{y+2}{\sqrt{2}\sqrt{x^2+(y+1)^2+1}}$$

$\ge \frac{\sqrt{3}}{2}$ より、両辺正なので2乗して整理：

$$4(y+2)^2 \ge 6\bigl(x^2+(y+1)^2+1\bigr)$$ $$4y^2+16y+16 \ge 6x^2+6y^2+12y+12$$ $$6x^2+2y^2-4y-4 \le 0$$ $$6x^2+2(y-1)^2 \le 6$$ $$\boxed{x^2 + \frac{(y-1)^2}{3} \le 1}$$

これは中心 $(0,1)$、$x$ 方向の半径 $1$、$y$ 方向の半径 $\sqrt{3}$ の楕円の内部。

【図示】

求める範囲は、楕円 $x^2+\frac{(y-1)^2}{3} \le 1$ と $y \ge |x|$ の共通部分（原点を除く）。

楕円と $y = x$ の交点：$x^2+(x-1)^2/3=1$ → $4x^2-2x-2=0$ → $x = 1$ または $x = -1/2$。 $y \ge |x|$ を満たすのは $(1,1)$ のみ。

楕円と $y = -x$ の交点：同様に $(-1,1)$ のみ。

楕円 $x^2+\frac{(y-1)^2}{3} \le 1$ と $y \ge |x|$ の共通部分（原点を除く）。
境界は楕円の弧（$(1,1)$ から $(-1,1)$ まで上側）と線分 $y = |x|$（$(0,0)$ から $(1,1)$, $(-1,1)$）。

① 状況理解で分かること

  • 空間の角度条件を $xy$ 平面上の領域に落とす問題。

② 用いた定石

  • 「なす角の扱い」→ 内積と $\cos$ で不等式にする（定石集FGverより）

第 2 問

次の関数 $f(x)$ を考える。

$$f(x) = \int_0^1 \frac{|t-x|}{1+t^2}\,dt \quad (0 \le x \le 1)$$

(1) $0 < \alpha < \dfrac{\pi}{4}$ を満たす実数 $\alpha$ で、$f'(\tan\alpha)=0$ となるものを求めよ。

(2) (1) で求めた $\alpha$ に対し、$\tan\alpha$ の値を求めよ。

(3) 関数 $f(x)$ の区間 $0 \le x \le 1$ における最大値と最小値を求めよ。必要ならば、$0.69 < \log 2 < 0.7$ であることを用いてよい。

方針の立て方

① 状況理解で分かること

  • $f(x) = \int_0^1 \frac{|t-x|}{1+t^2}\,dt$ は $0 \le x \le 1$ で定義。$|t-x|$ の絶対値を外すために $t = x$ で分割する。
  • 微分して $f'(x) = 0$ を求め、$\tan\alpha$ の形で表す。$1/(1+t^2)$ の原始関数が $\arctan t$ であることが鍵。

② 用いた定石

特になし

(1) の解法

$0 \le x \le 1$ のとき、$|t-x|$ を $t = x$ で分割：

$$f(x) = \int_0^x \frac{x-t}{1+t^2}\,dt + \int_x^1 \frac{t-x}{1+t^2}\,dt$$

$x$ で微分する（積分と微分の順序交換）：

$$f'(x) = \int_0^x \frac{1}{1+t^2}\,dt - \int_x^1 \frac{1}{1+t^2}\,dt = \arctan x - \left(\frac{\pi}{4} - \arctan x\right) = 2\arctan x - \frac{\pi}{4}$$

$f'(\tan\alpha) = 0$ とすると：

$$2\arctan(\tan\alpha) - \frac{\pi}{4} = 0 \quad \Longrightarrow \quad 2\alpha = \frac{\pi}{4}$$
$$\alpha = \frac{\pi}{8}$$

(2) の解法

半角の公式 $\tan\frac{\theta}{2} = \frac{\sin\theta}{1+\cos\theta}$ を $\theta = \frac{\pi}{4}$ に適用：

$$\tan\frac{\pi}{8} = \frac{\sin(\pi/4)}{1+\cos(\pi/4)} = \frac{\frac{\sqrt{2}}{2}}{1+\frac{\sqrt{2}}{2}} = \frac{\sqrt{2}}{2+\sqrt{2}} = \frac{\sqrt{2}(2-\sqrt{2})}{(2+\sqrt{2})(2-\sqrt{2})} = \frac{2\sqrt{2}-2}{2}$$
$$\tan\alpha = \sqrt{2}-1$$

(3) の解法

$f'(x) = 2\arctan x - \pi/4$。$\arctan x$ は単調増加なので、$f'(x) = 0$ の解は $x = \sqrt{2}-1$ のみ。

  • $x < \sqrt{2}-1$ で $f'(x) < 0$（単調減少）
  • $x > \sqrt{2}-1$ で $f'(x) > 0$（単調増加）

よって最小値は $x = \sqrt{2}-1$、最大値は端点 $x = 0$ または $x = 1$。

端点の値：

$$f(0) = \int_0^1 \frac{t}{1+t^2}\,dt = \left[\frac{1}{2}\ln(1+t^2)\right]_0^1 = \frac{\ln 2}{2}$$ $$f(1) = \int_0^1 \frac{1-t}{1+t^2}\,dt = \arctan 1 - \frac{\ln 2}{2} = \frac{\pi}{4} - \frac{\ln 2}{2}$$

$0.69 < \ln 2 < 0.7$ より $f(0) \approx 0.347$、$f(1) \approx 0.436$。よって $f(1)> f(0)$。

最小値：$x = \sqrt{2}-1$ での値。$f(x) = 2x\arctan x - \ln(1+x^2) + \frac{\ln 2}{2} - \frac{\pi x}{4}$ より、

$x = \sqrt{2}-1$, $\arctan x = \pi/8$ を代入すると $2x\arctan x = \frac{(\sqrt{2}-1)\pi}{4}$, $\frac{\pi x}{4} = \frac{(\sqrt{2}-1)\pi}{4}$ で打ち消し合い：

$$f(\sqrt{2}-1) = -\ln(4-2\sqrt{2}) + \frac{\ln 2}{2} = \ln\frac{\sqrt{2}+1}{2}$$
最大値：$f(1) = \dfrac{\pi}{4} - \dfrac{\ln 2}{2}$（$x = 1$）
最小値：$f(\sqrt{2}-1) = \ln\dfrac{\sqrt{2}+1}{2}$（$x = \sqrt{2}-1$）

方針の立て方のまとめ（第2問）

① 状況理解で分かること

  • $|t-x|$ の絶対値を外すために $t = x$ で積分を分割。微分すると $\arctan$ が現れる。
  • $f'(x) = 0$ が $\arctan$ の方程式になり、$\alpha = \pi/8$ → 半角公式で $\tan\alpha$ が求まる。
  • 端点比較には $\ln 2$ の評価が必要。

② 用いた定石

  • 「複数設問 → すべて誘導だと思え」（定石集FGver「誘導利用」より）

第 3 問

座標平面上を次の規則 (i), (ii) に従って1秒ごとに動く点 P を考える。

(i) 最初に、P は点 $(2,1)$ にいる。

(ii) ある時刻で P が $(a,b)$ にいるとき、その1秒後に P は

  • 確率 $\frac{1}{3}$ で $x$ 軸に関して対称な点 $(a,-b)$
  • 確率 $\frac{1}{3}$ で $y$ 軸に関して対称な点 $(-a,b)$
  • 確率 $\frac{1}{6}$ で直線 $y=x$ に関して対称な点 $(b,a)$
  • 確率 $\frac{1}{6}$ で直線 $y=-x$ に関して対称な点 $(-b,-a)$

(1) P がとりうる点の座標をすべて求めよ。

(2) $n$ を正の整数とする。最初から $n$ 秒後に P が点 $(2,1)$ にいる確率と、点 $(-2,-1)$ にいる確率は等しいことを示せ。

(3) $n$ を正の整数とする。最初から $n$ 秒後に P が点 $(2,1)$ にいる確率を求めよ。

方針の立て方

① 状況理解で分かること

  • 4つの操作はすべて「対称変換」（反射）であり、2回行うと元に戻る。
  • $(2,1)$ から到達可能な点を実験で列挙すると、座標の成分の絶対値 $\{1,2\}$ が保存される → 8点で閉じる。
  • (2) は原点対称なペア（$(2,1)$ と $(-2,-1)$ など）に着目すると、遷移先が同じ集合になっている → 帰納法で示せる見込み。

② 用いた定石

「場合の数の問題はどんな時も書き出しなどの実験からスタートする」（定石集FGverより）
「確率漸化式の立式 → まずはすべての状態を把握し遷移図を書く」（定石集FGverより）

(1) の解法

$(2,1)$ に4つの操作を適用：$(2,-1)$, $(-2,1)$, $(1,2)$, $(-1,-2)$。 これらにさらに操作を適用していくと、到達可能な点は以下の8点で閉じる：

$(\pm 2,\, \pm 1)$ および $(\pm 1,\, \pm 2)$（計8点）

（確認：任意の8点から4操作を施しても、8点の中に留まる。）

(2) の解法

【鍵となる観察】

8点を原点対称のペアに分ける：$(2,1) \leftrightarrow (-2,-1)$、$(2,-1) \leftrightarrow (-2,1)$、$(1,2) \leftrightarrow (-1,-2)$、$(1,-2) \leftrightarrow (-1,2)$。

各ペアの2点から遷移先の集合と確率が完全に同じであることを確認する：

出発点 $x$軸対称（$\frac{1}{3}$） $y$軸対称（$\frac{1}{3}$） $y=x$対称（$\frac{1}{6}$） $y=-x$対称（$\frac{1}{6}$）
$(2,1)$ $(2,-1)$ $(-2,1)$ $(1,2)$ $(-1,-2)$
$(-2,-1)$ $(-2,1)$ $(2,-1)$ $(-1,-2)$ $(1,2)$

$(2,1)$ からの行き先は $\{(2,-1),\, (-2,1),\, (1,2),\, (-1,-2)\}$。$(-2,-1)$ からの行き先も同じ集合で同じ確率（行の中で入れ替わっているだけ）。

他の3ペアでも同様のことが成り立つ（各自確認可）。

【帰納法による証明】

「$n$ 秒後に各点にいる確率は、原点対称なペアで等しい」…（$\star$） を示す。

$n = 1$ のとき：$P$ は $(2,1)$ から出発するので、1秒後の確率は：

  • $(2,-1)$：$\frac{1}{3}$、$(-2,1)$：$\frac{1}{3}$　← ペアで等しい ✓
  • $(1,2)$：$\frac{1}{6}$、$(-1,-2)$：$\frac{1}{6}$　← ペアで等しい ✓
  • その他：$0$　← ペアで等しい ✓

$n \to n+1$ の帰納ステップ：（$\star$）が $n$ で成り立つと仮定する。 上の表から、ペアの2点は同じ確率の集合に遷移する。よって $n+1$ 秒後も各ペア内の確率は等しい。

（$\star$）が全ての $n \ge 1$ で成り立つ。特に $(2,1)$ と $(-2,-1)$ のペアで：

$n$ 秒後に $(2,1)$ にいる確率 $=$ $n$ 秒後に $(-2,-1)$ にいる確率 $\qquad \square$

(3) の解法

(2) と同様の対称性により、8状態は4つのペアに分かれ、各ペア内で確率が等しい：

  • $p_n = P(X_n = (2,1)) = P(X_n = (-2,-1))$
  • $q_n = P(X_n = (2,-1)) = P(X_n = (-2,1))$
  • $r_n = P(X_n = (1,2)) = P(X_n = (-1,-2))$
  • $u_n = P(X_n = (1,-2)) = P(X_n = (-1,2))$

正規化条件：$2(p_n+q_n+r_n+u_n) = 1$。

遷移を調べると（$n \ge 1$ のとき）：

$$p_{n+1} = \frac{2}{3}q_n + \frac{1}{3}r_n, \quad q_{n+1} = \frac{2}{3}p_n + \frac{1}{3}u_n$$ $$r_{n+1} = \frac{1}{3}p_n + \frac{2}{3}u_n, \quad u_{n+1} = \frac{1}{3}q_n + \frac{2}{3}r_n$$

【簡約化】$P_n = p_n+u_n$, $Q_n = q_n+r_n$ とおくと：

$$P_{n+1} = Q_n, \quad Q_{n+1} = P_n$$

初期値：$p_1 = 0$, $q_1 = 1/3$, $r_1 = 1/6$, $u_1 = 0$。$P_1 = 0$, $Q_1 = 1/2$。

よって $n$ 偶数のとき $P_n = 1/2$, $Q_n = 0$。$n$ 奇数のとき $P_n = 0$, $Q_n = 1/2$。

$\alpha_n = p_n - u_n$, $\beta_n = q_n - r_n$ とおくと：

$$\alpha_{n+1} = \frac{1}{3}\beta_n, \quad \beta_{n+1} = \frac{1}{3}\alpha_n$$

初期値：$\alpha_1 = 0$, $\beta_1 = 1/6$。$\alpha_{2k} = \frac{1}{2 \cdot 9^k}$（帰納法で確認可）。

$n$ が偶数（$n = 2k$）のとき：$p_n = \frac{P_n+\alpha_n}{2} = \frac{1/2 + 1/(2 \cdot 9^k)}{2} = \frac{9^k+1}{4 \cdot 9^k} = \frac{3^n+1}{4 \cdot 3^n}$。

$n$ が奇数のとき：$P_n = 0$, $\alpha_n = 0$ より $p_n = 0$。

$$P(X_n = (2,1)) = \begin{cases} \dfrac{3^n+1}{4 \cdot 3^n} & (n \text{ が偶数}) \\[8pt] 0 & (n \text{ が奇数}) \end{cases}$$

方針の立て方のまとめ（第3問）

① 状況理解で分かること

  • 4操作で閉じる8状態を列挙。原点対称なペアで遷移先が同じ → 帰納法でペアの確率が等しい。
  • 4ペアの確率で漸化式を立て、和と差に分解すると解ける。
  • $n$ 奇数で $p_n = 0$ は、各操作が必ず状態を変える（偶奇性がある）ことの反映。

② 用いた定石

  • 「場合の数はどんな時も実験からスタート」（定石集FGverより）
  • 「確率漸化式 → すべての状態を把握し遷移図を書く」（定石集FGverより）

第 4 問

$f(x) = -\dfrac{\sqrt{2}}{4}x^2 + 4\sqrt{2}$ とおく。$0 < t < 4$ を満たす実数 $t$ に対し、座標平面上の点 $(t, f(t))$ を通り、 この点において放物線 $y=f(x)$ と共通の接線を持ち、$x$ 軸上に中心を持つ円を $C_t$ とする。

(1) 円 $C_t$ の中心の座標を $(c(t), 0)$、半径を $r(t)$ とおく。$c(t)$ と $\{r(t)\}^2$ を $t$ の整式で表せ。

(2) 実数 $a$ は $0 < a < f(3)$ を満たすとする。円 $C_t$ が点 $(3, a)$ を通るような実数 $t$ は $0 < t < 4$ の範囲にいくつあるか。

方針の立て方

① 状況理解で分かること

  • 円と放物線が $(t, f(t))$ で共通接線をもつ → 円の中心は接点における法線上にある。
  • 法線が $x$ 軸と交わる点が中心 $(c(t), 0)$。

② 用いた定石

「接線の問題 → 接点をおく → 接線の式が立つ」（IIB定石リストより）
「円と放物線が接する条件」→ 法線が中心を通る（定石集FGverより）

(1) の解法

$f'(t) = -\frac{\sqrt{2}}{2}t$。接点 $(t, f(t))$ での法線の傾きは $-1/f'(t) = \frac{\sqrt{2}}{t}$（$t \neq 0$）。

法線の方程式：$y - f(t) = \frac{\sqrt{2}}{t}(x - t)$。$y = 0$ を代入して $c(t)$ を求める：

$$-f(t) = \frac{\sqrt{2}}{t}(c - t) \quad \Longrightarrow \quad c = t - \frac{t \cdot f(t)}{\sqrt{2}}$$

$f(t) = -\frac{\sqrt{2}}{4}t^2 + 4\sqrt{2}$ より $\frac{t \cdot f(t)}{\sqrt{2}} = -\frac{t^3}{4} + 4t$ なので：

$$c(t) = t + \frac{t^3}{4} - 4t = \frac{t^3 - 12t}{4} = \frac{t(t^2-12)}{4}$$

$r^2 = (t-c)^2 + f(t)^2$。$t - c = \frac{t(16-t^2)}{4}$、$f(t) = \frac{\sqrt{2}(16-t^2)}{4}$ なので：

$$r^2 = \frac{(16-t^2)^2}{16}\left(t^2 + 2\right)$$
$$c(t) = \frac{t(t^2-12)}{4}, \qquad \{r(t)\}^2 = \frac{(16-t^2)^2(t^2+2)}{16}$$

(2) の解法

【方程式の立式】

円 $C_t$ が $(3,a)$ を通る条件：$(3-c(t))^2 + a^2 = r(t)^2$。

これを「$a^2 = (\cdots)$」の形に整理して、右辺を $t$ の関数とみるのが見通しのよい方針。

$$a^2 = r(t)^2 - (3-c(t))^2 =: F(t)$$

$0 < a < f(3) = \frac{7\sqrt{2}}{4}$ のもとで、$F(t) = a^2$ を満たす $t \in (0,4)$ の個数を調べる。

【$F(t)$ の計算】

$3 - c(t) = \frac{-t^3+12t+12}{4}$ なので：

$$F(t) = \frac{(16-t^2)^2(t^2+2)}{16} - \frac{(-t^3+12t+12)^2}{16}$$

分子を展開・整理すると（4次式に整理される）：

$$F(t) = \frac{-3t^4 + 12t^3 + 24t^2 - 144t + 184}{8}$$

【$F(t)$ の増減を調べる】

$$F'(t) = \frac{-12t^3 + 36t^2 + 48t - 144}{8} = \frac{-12(t-2)(t-3)(t+2)}{8}$$

$0 < t < 4$ における符号変化：

$t$$0$$2$$3$$4$
$F'(t)$$-$$0$$+$$0$$-$
$F(t)$$23$$\searrow$$5$$\nearrow$$\frac{49}{8}$$\searrow$$-1$

各値の確認：$F(0) = \frac{184}{8} = 23$、$F(2) = \frac{40}{8} = 5$（極小）、$F(3) = \frac{49}{8}$（極大）、$F(4) = \frac{-8}{8} = -1$。

【グラフと水平線 $y = a^2$ の交点の個数】

$F(t)$ は $t \in (0,4)$ で「$23$ から下がって $5$ → 上がって $\frac{49}{8}$ → 下がって $-1$」の山谷構造。

$0 < a < f(3) = \frac{7\sqrt{2}}{4}$ より $0 < a^2 < \frac{49}{8}$。水平線 $y = a^2$ との交点数は：

  • $0 < a^2 < 5$（$0 < a < \sqrt{5}$）：水平線は極小値 $5$ より下。減少区間 $(0,2)$ と増加区間 $(2,3)$ では $F \ge 5 > a^2$ なので交わらない。減少区間 $(3, 4)$ で $F$ が $\frac{49}{8}$ から $-1$ に下がる途中で1回交わる。→ 1個
  • $a^2 = 5$（$a = \sqrt{5}$）：極小値にちょうど接触して $t = 2$ で1回。$(3,4)$ で1回。→ 2個
  • $5 < a^2 < \frac{49}{8}$（$\sqrt{5} < a < \frac{7\sqrt{2}}{4}$）：減少区間 $(0,2)$ で1回、増加区間 $(2,3)$ で1回、減少区間 $(3,4)$ で1回。→ 3個
  • $0 < a < \sqrt{5}$ のとき：1個
  • $a = \sqrt{5}$ のとき：2個
  • $\sqrt{5} < a < \frac{7\sqrt{2}}{4}$ のとき：3個

① 状況理解で分かること

第 5 問

座標空間内に3点 $A(1,0,0)$, $B(0,1,0)$, $C(0,0,1)$ をとり、$D$ を線分 $AC$ の中点とする。 三角形 $ABD$ の周および内部を $x$ 軸のまわりに1回転させて得られる立体の体積を求めよ。

方針の立て方

① 状況理解で分かること

  • $D = (1/2, 0, 1/2)$。三角形 $ABD$ の頂点：$A(1,0,0)$, $B(0,1,0)$, $D(1/2, 0, 1/2)$。
  • $x$ 軸回転の断面を考える。$x = c$（定数）で切ると、三角形の断面は線分（三角形と平面 $x = c$ の交わり）。 この線分を $x$ 軸のまわりに回転すると、線分上の各点は $x$ 軸からの距離 $\sqrt{y^2+z^2}$ を半径とする円を描く。 線分上で $x$ 軸からの距離は連続に変化するので、回転体の断面は最小距離 $r$ から最大距離 $R$ までの円環（$r^2 \le y^2+z^2 \le R^2$）になる。
  • 頂点 $A$, $D$ は $x$ 軸上（$y = z = 0$）にあるが、$B$ は $x$ 軸の外（距離 $1$）。 よって三角形は $x$ 軸と交わるが $x$ 軸を含まないので、$0 < x < 1$ では $r > 0$（円板ではなく円環）。

② 用いた定石

「体積 → 断面で考える」（定石集FGver「§体積」より）

解法

【各辺の $x$ 軸からの距離】

  • 辺 $AB$：$(1-t, t, 0)$, $t \in [0,1]$。$x = 1-t$ で距離 $= t = 1-x$。範囲 $x \in [0,1]$。
  • 辺 $AD$：$(1-t/2, 0, t/2)$, $t \in [0,1]$。$x = 1-t/2$ で距離 $= t/2 = 1-x$。範囲 $x \in [1/2,1]$。
  • 辺 $BD$：$(t/2, 1-t, t/2)$, $t \in [0,1]$。$x = t/2$ で距離 $= \sqrt{(1-2x)^2+x^2}$。範囲 $x \in [0,1/2]$。

【断面解析】

三角形内部の点で $x$ 座標が一定のものは、辺上の2点を結ぶ線分。 $x$ 軸からの距離の最大値が外半径 $R$、最小値が内半径 $r$。

外半径：$R(x) = 1-x$（辺 $AB$ 上。辺 $AB$ は $z = 0$ かつ $y = 1-x > 0$ なので、他の辺より $x$ 軸から遠い）。

内半径の求め方：

$x$ を固定すると、三角形の断面は辺上の2点を結ぶ線分。$x \in [0, 1/2]$ では辺 $AB$ 上の点 $(x,\,1-x,\,0)$ と辺 $BD$ 上の点 $(x,\,1-2x,\,x)$ を結ぶ線分になる。

この線分上の点をパラメータ $s \in [0,1]$ で表すと：

$$(y,\,z) = (1-s)(1-x,\, 0) + s(1-2x,\, x) = (1-x-sx,\; sx)$$

$x$ 軸からの距離の2乗は：

$$\rho(s)^2 = (1-x-sx)^2 + (sx)^2 = (1-x)^2 - 2(1-x)sx + s^2 x^2 + s^2 x^2$$ $$= (1-x)^2 - 2(1-x)xs + 2x^2 s^2$$

$s$ で微分して最小点を求める：

$$\frac{d(\rho^2)}{ds} = -2(1-x)x + 4x^2 s = 2x\bigl(-1+x+2xs\bigr) = 0$$

$x > 0$ なので $s = \frac{1-x}{2x}$。

この $s$ が $[0,1]$ に入るかどうかで場合分け：

  • $s \le 1 \iff \frac{1-x}{2x} \le 1 \iff 1-x \le 2x \iff x \ge \frac{1}{3}$： 最小点は線分の内部にある。代入すると $y = \frac{1-x}{2}$, $z = \frac{1-x}{2}$ で： $$r(x)^2 = \frac{(1-x)^2}{4} + \frac{(1-x)^2}{4} = \frac{(1-x)^2}{2}$$
  • $s > 1 \iff x < \frac{1}{3}$： 最小点は $s = 1$（辺 $BD$ 上の端点）で達成。$y = 1-2x$, $z = x$ なので： $$r(x)^2 = (1-2x)^2 + x^2 = 1-4x+5x^2$$

$x \in [1/2, 1]$ では断面が辺 $AB$ と辺 $AD$ を結ぶ線分になり、同様の計算で $r(x)^2 = (1-x)^2/2$（$x \ge 1/3$ の場合と同じ式）。

【体積計算】

$$V = \pi\int_0^1 \bigl[R(x)^2 - r(x)^2\bigr]\,dx$$ $$= \pi\int_0^{1/3}\bigl[(1-x)^2 - (1-4x+5x^2)\bigr]\,dx + \pi\int_{1/3}^1 \frac{(1-x)^2}{2}\,dx$$

第1積分：$(1-x)^2-(1-4x+5x^2) = 2x-4x^2$。

$$\int_0^{1/3}(2x-4x^2)\,dx = \left[x^2-\frac{4x^3}{3}\right]_0^{1/3} = \frac{1}{9}-\frac{4}{81} = \frac{5}{81}$$

第2積分：

$$\int_{1/3}^1 \frac{(1-x)^2}{2}\,dx = \left[-\frac{(1-x)^3}{6}\right]_{1/3}^1 = 0+\frac{(2/3)^3}{6} = \frac{8}{162} = \frac{4}{81}$$
$$V = \pi\left(\frac{5}{81}+\frac{4}{81}\right) = \frac{\pi}{9}$$

方針の立て方のまとめ（第5問）

① 状況理解で分かること

  • 回転体の断面は円環。各辺上の点の $x$ 軸からの距離を $x$ の関数で表す。
  • 内半径の最小点が辺上か内部かで場合分け（$x = 1/3$ が境界）。

② 用いた定石

  • 「体積 → 断面で考える」（定石集FGverより）

第 6 問

2以上の整数で、1とそれ自身以外に正の約数を持たない数を素数という。以下の問いに答えよ。

(1) $f(x) = x^3+10x^2+20x$ とする。$f(n)$ が素数となるような整数 $n$ をすべて求めよ。

(2) $a, b$ を整数の定数とし、$g(x) = x^3+ax^2+bx$ とする。$g(n)$ が素数となるような整数 $n$ の個数は3個以下であることを示せ。

方針の立て方

① 状況理解で分かること

  • $f(x) = x(x^2+10x+20)$、$g(x) = x(x^2+ax+b)$ と因数分解できる。
  • 整数の積が素数 → 一方が $\pm 1$、他方が $\pm p$（IA定石リスト「素数の扱い」①）。
  • (1) は具体的に場合分け。(2) は各場合の解の個数を数える。

② 用いた定石

「整数 基本方針…まずは実験から」「$p = (\text{整数})\times(\text{整数})$ と変形し、素数の約数は $\pm 1, \pm p$ しかないことを利用」（IA定石リストより）

(1) の解法

$f(n) = n(n^2+10n+20)$。$f(n)$ が素数 $(\ge 2)$ となるには：

【$n > 0$ の場合】$n \ge 1$ かつ $n^2+10n+20 \ge 31 > 0$。積が素数 → $n = 1$。

$f(1) = 1 \cdot 31 = 31$（素数 ✓）。

【$n < 0$ の場合】$f(n) > 0$ となるには、$n < 0$ かつ $n^2+10n+20 < 0$ が必要。

$n^2+10n+20 = (n+5)^2-5 < 0$ → $n \in \{-7,-6,-5,-4,-3\}$。

$n$ $n^2+10n+20$ $f(n)$ 素数？
$-3$ $-1$ $3$ ✓
$-4$ $-4$ $16$ ✗
$-5$ $-5$ $25$ ✗
$-6$ $-4$ $24$ ✗
$-7$ $-1$ $7$ ✓
$$n = -7,\; -3,\; 1$$

(2) の解法

$g(n) = n(n^2+an+b)$ が素数のとき、$|n| = 1$ または $|n^2+an+b| = 1$。

Case A：$n = 1$ → $g(1) = 1+a+b$。素数なら1個。

Case B：$n = -1$ → $g(-1) = -(1-a+b) = a-b-1$。素数なら1個。

Case C：$n^2+an+b = 1$ → 2次方程式。整数解は高々2個。素数になるには $g(n) = n$ なので $n \ge 2$。

Case D：$n^2+an+b = -1$ → 2次方程式。整数解は高々2個。素数になるには $g(n) = -n$ なので $n \le -2$。

【Case C と Case D は両立しない】

Case C に整数解 $n_1 \ge 2$、Case D に整数解 $m_1 \le -2$ があると仮定する。

$$n_1^2+an_1+b = 1, \quad m_1^2+am_1+b = -1$$

辺々引くと：$(n_1-m_1)(n_1+m_1+a) = 2$。

$n_1 \ge 2$, $m_1 \le -2$ より $n_1-m_1 \ge 4$。しかし $4$ は $2$ を割らない。矛盾。

よって C, D の一方のみが解をもつ。解の個数は高々2。A, B は各1個。

また C が2解をもつとき B は素数を与えない（計算で確認可）。D が2解のときも同様に A のみ。

$g(n)$ が素数となる $n$ の個数は $2 + 1 = 3$ 以下。$\qquad \square$

等号の例：$a = -5$, $b = 7$ のとき $g(x) = x(x^2-5x+7)$。$g(1)=3$, $g(2)=2$, $g(3)=3$。3個。

方針の立て方のまとめ（第6問）

① 状況理解で分かること

  • $g(n) = n(n^2+an+b)$ と因数分解 → 素数なら一方が $\pm 1$。
  • 4つの場合（$n=\pm 1$, $n^2+an+b = \pm 1$）に分類し、個数を数える。
  • Case C と D が両立しないことが上界 $3$ の鍵。

② 用いた定石

  • 「整数は実験からスタート」（IA定石リストより）
  • 「$p = (\text{整数})\times(\text{整数})$ → 約数は $\pm 1, \pm p$」（IA定石リスト「素数の扱い」①より）
